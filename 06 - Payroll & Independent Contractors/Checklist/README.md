# Payroll & Independent Contractors

## Overview
This repository contains the control framework, procedures, and operational tools used to manage **payroll** and **independent contractor** payments within a structured, audit-ready finance environment.

The objective is to ensure:

- accuracy of payroll calculations  
- completeness of liabilities  
- appropriate authorisation prior to payment  
- correct accounting treatment  
- full traceability between source documents, reconciliations, and the general ledger  

The framework reflects strong financial governance and segregation of duties expected within professional finance functions.

## Scope of Control

### Payroll
- Gross-to-net validation  
- PAYE, UIF, and SDL calculation checks  
- Variance analysis  
- Liability reconciliation to the GL  
- Payment approval and release  
- Post-payment verification  

### Independent Contractors
- Invoice and rate validation  
- Contract or service confirmation  
- Expense and GL coding  
- Accrual assessment where applicable  
- Approval workflow  
- Settlement and ageing monitoring
