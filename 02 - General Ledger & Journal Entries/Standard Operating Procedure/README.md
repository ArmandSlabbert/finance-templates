## Overview
This directory contains the official Standard Operating Procedure governing how journal entries are prepared, supported, reviewed, approved, and posted to the general ledger.

The aim is to:

- protect financial integrity  
- ensure consistency  
- maintain audit defensibility  
- prevent unauthorised or unsupported postings  

This is a core financial control environment document.

---

## What This SOP Covers
The procedure defines mandatory rules for:

- Journal preparation standards  
- Minimum supporting documentation  
- Reviewer responsibilities  
- Approval hierarchy  
- Posting permissions  
- Reversals and corrections  
- Audit trail requirements  

It applies to **all entities**, **all reporting periods**, and both **manual and automated** journals.
